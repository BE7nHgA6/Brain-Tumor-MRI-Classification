
# Brain Tumor MRI Image Classification

## Project Overview

This project aims to classify brain MRI images into four categories: **Glioma, Meningioma, Pituitary Tumor, and No Tumor** using Deep Learning techniques. The objective is to assist in the early detection and classification of brain tumors from MRI scans.

## Technologies Used

* Python
* TensorFlow / Keras
* CNN (Convolutional Neural Network)
* EfficientNetB0 (Transfer Learning)
* Streamlit
* NumPy, Pandas, Matplotlib, Seaborn

## Project Workflow

1. Data Collection and Exploration
2. Image Preprocessing and Augmentation
3. Custom CNN Model Development
4. Transfer Learning using EfficientNetB0
5. Model Evaluation using Accuracy, Precision, Recall, F1-Score, and Confusion Matrix
6. Model Comparison
7. Streamlit Web Application Deployment

## Results

* Custom CNN Accuracy: **49%**
* EfficientNetB0 Accuracy: **82.52%**
* EfficientNetB0 achieved the best performance and was selected as the final deployment model.

## Features

* Upload Brain MRI Images
* Real-Time Tumor Classification
* Confidence Score Display
* User-Friendly Streamlit Interface

##Conclusion

Transfer Learning with EfficientNetB0 significantly improved classification performance compared to the Custom CNN model, making it a reliable solution for brain tumor MRI image classification.
